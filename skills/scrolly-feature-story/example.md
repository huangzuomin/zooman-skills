# 示例输入：温州新闻网产品和服务手册 2026

视觉方向：Institutional + Editorial + Data Story

目标受众：政企客户、项目决策者、内部经营团队

传播目标：把温州新闻网的内容传播、平台运营、技术开发、数据服务、活动执行和 AI 应用能力，转化为清晰可信、便于对外推介的滚动叙事产品手册。

## 期望体验

- 首屏建立“城市智媒服务平台”的品牌定位。
- 七大能力板块使用 sticky matrix 或能力图谱表达。
- 产品项使用可扫描但不模板化的分组卡片。
- 案例库突出类型与核心价值。
- 服务包突出客户画像、包含内容、适用场景。
- 结束页用大字收束“从内容到系统、从传播到数据、从项目到产品、从经验到能力”。
