# Visual Style Library

Use this reference after choosing page templates and before writing layout/CSS. A visual style defines the page's editorial personality, not just colors.

## Style Decision Protocol

Before implementation, state:

```text
Primary visual style:
Secondary visual style:
Tone:
Typography:
Color system:
Layout motifs:
Asset style:
Forbidden motifs:
```

## Styles

### newsroom-bureau

Best for:

- Media product manuals.
- Newsroom capability systems.
- Case portfolios.
- Editorial productization stories.

Tone:

- Confident.
- Editorial.
- Operational.
- Evidence-backed.

Typography:

- Strong Chinese display heading.
- Compact metadata labels.
- Editorial captions.
- Dossier-style numbering.

Color:

- Warm paper or quiet neutral background.
- Deep ink text.
- One editorial accent such as red, vermilion, or signal orange.
- Optional civic green/blue in restrained supporting roles.

Layout motifs:

- Product dossier sheets.
- Assignment boards.
- Archive rails.
- Clipping walls.
- Stamp-like labels.
- Evidence strips.

Assets:

- Newsroom scenes.
- Article/screenshot excerpts.
- Product taxonomy diagrams.
- Case clippings.
- Workflow boards.

Avoid:

- Startup SaaS cards.
- Generic tech gradients.
- Decorative icons without editorial meaning.

### civic-atlas

Best for:

- City service stories.
- Local media public-service narratives.
- Government-adjacent institutional pages.
- Regional collaboration.

Tone:

- Calm.
- Public-minded.
- Clear.
- Trustworthy.

Typography:

- Stable hierarchy.
- Map labels and field-note captions.
- Clear section numbering.

Color:

- Civic neutrals.
- Muted green, blue, clay, or ink.
- Accent used for active regions or service highlights.

Layout motifs:

- Map sheets.
- Coordinate labels.
- Regional grids.
- Field notes.
- Documentary image bands.

Assets:

- Maps.
- Regional outlines.
- Public-service scenes.
- City imagery.

Avoid:

- Random map dots.
- Overly official portal styling.
- Decorative geography that says nothing.

### data-dossier

Best for:

- Data stories.
- Monitoring and dashboards.
- Research reports.
- Trend evidence.

Tone:

- Precise.
- Analytical.
- Low-noise.
- Transparent.

Typography:

- Clear chart labels.
- Source notes.
- Numeric hierarchy.
- Evidence tabs.

Color:

- Neutral base.
- Limited analytic palette.
- Strong highlight color for selected data.

Layout motifs:

- Annotated charts.
- Report sheets.
- Tabs.
- Evidence callouts.
- Timeline rails.

Assets:

- Real charts.
- Tables.
- Dashboards.
- Report pages.

Avoid:

- Fake charts.
- Decorative data marks.
- Excessive animation.

### magazine-feature

Best for:

- Longform profiles.
- Cultural stories.
- Investigative features.
- Human-centered narratives.

Tone:

- Literary.
- Immersive.
- Reflective.

Typography:

- Big editorial titles.
- Narrow reading columns.
- Pull quotes.
- Captions.

Color:

- Image-led.
- Quiet text surfaces.
- Accent sparingly.

Layout motifs:

- Full-bleed photography.
- Text/image alternation.
- Pullquote spreads.
- Chapter openers.

Assets:

- Strong photography.
- Portraits.
- Documentary scenes.

Avoid:

- Product grids.
- Dense dashboard components.

### event-stage

Best for:

- Events.
- Broadcast recaps.
- Campaigns.
- Competitions.

Tone:

- Energetic.
- Time-based.
- Participatory.

Typography:

- Bold opening.
- Program labels.
- Short sections.

Color:

- More vivid than institutional styles.
- Still constrained to avoid poster clutter.

Layout motifs:

- Stage/program rundown.
- Live control room.
- Photo strips.
- Schedule rails.

Assets:

- Event photography.
- Venue imagery.
- Live screenshots.
- Participant moments.

Avoid:

- Slow institutional pacing.
- Overcrowded celebratory visuals.

### institutional-report

Best for:

- Annual reports.
- Official summaries.
- Public service impact.
- Organizational review.

Tone:

- Stable.
- Accountable.
- Clear.
- Measured.

Typography:

- Report headings.
- Section numbers.
- Clear table/metric labels.

Color:

- Quiet neutrals.
- Official brand accent.
- High readability.

Layout motifs:

- Report pages.
- Milestone rails.
- Metric summaries.
- Evidence spreads.

Assets:

- Project images.
- Official documents.
- Charts.
- Timeline evidence.

Avoid:

- Celebration-only poster style.
- Dense bureaucratic pages.

## Wenzhou News Style

Recommended:

```text
Primary visual style: newsroom-bureau
Secondary visual style: civic-atlas
Data sections: data-dossier
Tone: confident, civic, editorial, operational
Visual metaphor: a newsroom turns scattered assignments and capabilities into a city media service product archive
```

The design should feel like an editorial product bureau: structured, local, useful, and media-native.
